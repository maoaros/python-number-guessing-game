# Guessing Number Game (Using Python)

This project serves as an entry-level implementation designed to showcase clean code structuring, robust exception safety, and standard algorithmic workflows in Python.

## Features

- **Dynamic Number Generation:** Uses Python's standard library `random` module to select a unique number every game.
- **Robust Input Validation:** Utilizes `try-except` blocks to intercept non-integer data streams, preventing execution failure.
- **Metric Tracker:** Tallies user decision cycles (attempts) dynamically to log efficiency metrics upon success.

## Setup and Execution

1. Clone or Download the Project: Create a file named guessing_game.py and paste the game code inside it.

2. Open Terminal / Command Prompt: Navigate to the directory where your file is saved:

  ```Bash
  cd path/to/your/project-folder
  ```

3. Run the Game: Execute the script using Python:

  ```Bash
  python guessing_game.py
  ```
